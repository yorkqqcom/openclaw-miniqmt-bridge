# 基金资金流向 — 宏观上下文分析方法

## 用途

为研究员辩论提供宏观资金面视角。小盘股选股组合的胜率与市场风格（大/小盘）高度相关。

## 核心方法

### 1. 中小盘ETF份额变动追踪

**目的**：判断当前市场风格倾向（大盘/小盘），辅助判断小盘股持仓的宏观风险。

**推荐监控标的：**

| ETF代码 | 名称 | 覆盖 | 意义 |
|:---|:---|:---:|:---|
| 512100.SH | 南方中证1000ETF | 小盘1000只 | 小盘资金风向标 |
| 159845.SZ | 华夏中证1000ETF | 小盘1000只 | 验证中证1000趋势 |
| 510500.SH | 南方中证500ETF | 中盘500只 | 中盘资金风向标 |
| 512510.SH | 中证500ETF | 中盘500只 | 验证中证500趋势 |

**查询方法：**
```python
# 份额变动趋势（4~5月区间）
get_fund_share_range(ts_code="512100.SH", start_date="2026-04-01", end_date="2026-05-06", limit=20)
# → items[0]=最新, items[-1]=最早
# → 计算: (items[0].fd_share - items[-1].fd_share) / items[-1].fd_share

# 日环比变动（更精细）
get_fund_share_change_range(ts_code="512100.SH", start_date="2026-04-20", end_date="2026-05-06", limit=10)
# → items[].delta_pct: 日环比
```

**判断标准：**
| 份额变动 | 信号 |
|:---:|:---|
| +10%~+40% | 小盘资金大幅流入 ✅ 利好小盘组合 |
| +3%~+10% | 温和流入，中性偏多 |
| -5%~+3% | 横盘，中性 |
| <-5% | 资金流出 ⚠️ 注意小盘风险 |

### 2. 个股权重基金持仓的限制

**关键事实：** 小盘股（流通市值<50亿）通常不在主流ETF的**前30大持仓中**。

| ETF类型 | 第30大重仓常见权重 | 覆盖特点 |
|:---|:---:|:---|
| 沪深300ETF | ~0.3%~1.0% | 只覆盖300只大盘 |
| 中证500ETF | ~0.2%~0.5% | 中盘500只 |
| 中证1000ETF | ~0.07%~0.15% | 第30名仅0.07%，小盘不易进入 |
| 行业ETF | 0.3%~3% | 行业内大市值优先 |

**影响：** 今日买入的小盘标的（广东建科、中机认检、德尔玛、广博股份）**不会出现在任何主流ETF的前30重仓中**。通过常规的 `get_fund_portfolio_report` 查不到它们的基金持仓。

**替代分析路径：**
- 用中小盘ETF总体份额趋势判断资金面大方向
- 用 `get_fund_portfolio_report` 搜非主流/行业主题ETF时，需要扩大limit到50+
- 对于光明乳业这种中盘消费股，检查消费/食品ETF需要查50-100名持仓

### 3. 全市场基金持仓反查

**工具限制：** 目前没有"给定股票→列出所有持有该股票的基金"的反查功能。

可用的近似方案：
```python
# 通过季报查某只基金是否持有我们的标的
get_fund_portfolio_report(ts_code="基金代码", end_date="2025-12-31", limit=30)
# → 检查 items[].symbol 是否匹配我们的标的

# 通过两期对比看个股的基金持股变化（需要先物化）
get_fund_holdings_diff_stock(old_period="2025-09-30", new_period="2025-12-31")
# → 但此表需要先运行 materialize_holdings_diff()，否则返回0条
```
